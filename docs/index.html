EPG maison

API EPG sport ouverte

Pour les chaînes sport sans guide fourni, KTV interroge automatiquement un service maison. Tu peux aussi l'appeler directement (JSON, CORS ouvert).

  • GET /sport — grille du jour de toutes les chaînes sport.
  • GET /epg?channel=beinsport1 — programme en cours + suivant d'une chaîne.
  • GET /health — état du service.
# Programme en cours et suivant de beIN Sports 1
curl "https://ktv-epg.khalilbenaz.workers.dev/epg?channel=beinsport1"

st / en = horodatages Unix (UTC), à convertir côté client. Noms normalisés : beinsport1, eurosport2, lequipe… Le parsing se corrige côté serveur, sans mettre à jour l'app.

Installer

Installation

macOS

Décompresse KTV-macos-arm64.zip, glisse KTV dans Applications, puis clic droit → Ouvrir (app non notarisée).

Si bloqué (macOS)

xattr -dr com.apple.quarantine "/Applications/KTV.app"

Windows

Décompresse KTV-windows-x64.zip et lance KTV.exe (SmartScreen → Informations complémentaires → Exécuter quand même).
